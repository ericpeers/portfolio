### Setup
* Create a python venv: python -m venv venv
* Enter it: source venv/bin/activate
* Install packages: pip install -r requirements.txt
* Run the scraper against your list of ETF's: python fidelity_scraper.py
* Be ready with your phone, not running the fidelity app, so that you can enter username/password, and then clean-open the app, to accept the login request.
